Judul: Struktur Data dan Algoritma dengan Python
Penulis: Michael T. Goodrich, Roberto Tamassia, Michael H. Goldwasser
Genre: Ilmu Komputer, Algoritma, Pemrograman
Penerbit: Wiley, 2016
Kata Kunci: Data, Struktur Data, Python, Algoritma

Isi:
Buku Struktur Data dan Algoritma dengan Python merupakan panduan komprehensif yang membahas konsep fundamental dalam pengorganisasian dan pemrosesan data secara efisien menggunakan bahasa Python. Buku ini menggabungkan teori struktur data klasik dengan pendekatan implementatif berbasis kode, menjadikannya sumber utama bagi mahasiswa dan pengembang perangkat lunak.

Topik utama yang dibahas mencakup:

1. Konsep Dasar Struktur Data  
   Buku ini menjelaskan definisi struktur data, hubungan antara kompleksitas waktu dan ruang, serta pentingnya memilih struktur data yang tepat dalam menyelesaikan masalah komputasi. Diperkenalkan notasi Big-O sebagai alat untuk menganalisis efisiensi algoritma.

2. Tipe Data Abstrak (Abstract Data Type / ADT)  
   Dibahas bagaimana ADT merepresentasikan perilaku logis dari struktur data tanpa memerhatikan implementasi internalnya. Contoh ADT yang dibahas meliputi Stack, Queue, List, dan Deque, beserta operasi dasar seperti push, pop, enqueue, dan dequeue.

3. Struktur Data Linear  
   Buku ini membahas array, linked list (singly dan doubly linked list), serta cara mengimplementasikannya di Python. Setiap struktur dilengkapi analisis kompleksitas operasi penyisipan, penghapusan, dan pencarian elemen.

4. Struktur Data Non-Linear  
   Pembahasan meliputi Tree dan Graph yang digunakan untuk merepresentasikan hubungan hierarkis dan jaringan kompleks. Termasuk implementasi binary search tree (BST), heap, AVL tree, dan graph traversal algorithms seperti Depth-First Search (DFS) dan Breadth-First Search (BFS).

5. Algoritma Pencarian dan Pengurutan (Searching & Sorting)  
   Buku ini menjelaskan berbagai algoritma penting seperti linear search, binary search, selection sort, insertion sort, merge sort, dan quick sort. Analisis perbandingan kompleksitas setiap algoritma membantu pembaca memahami performa pada berbagai ukuran data.

6. Hashing dan Struktur Data Asosiatif  
   Diperkenalkan konsep hash table dan dictionary sebagai struktur data efisien untuk pencarian cepat. Pembahasan meliputi fungsi hash, penanganan collision, dan penerapan dictionary Python yang berbasis open addressing.

7. Rekursi dan Analisis Kompleksitas  
   Buku ini menekankan penggunaan rekursi dalam penyelesaian masalah seperti tree traversal dan divide and conquer algorithms. Diberikan juga pendekatan sistematis untuk menganalisis kompleksitas algoritma rekursif menggunakan relasi rekursi.

8. Aplikasi Struktur Data dalam Dunia Nyata  
   Disajikan contoh penerapan struktur data pada berbagai bidang seperti sistem manajemen basis data, kompresi data, jaringan komputer, dan pemrosesan graf. Buku ini juga menekankan prinsip desain algoritma yang efisien dan dapat diskalakan.

Dengan penyajian yang sistematis, ilustrasi visual, serta contoh implementasi langsung dalam Python, buku ini membantu pembaca memahami hubungan erat antara teori dan praktik. Pembaca tidak hanya belajar cara menggunakan struktur data, tetapi juga mengapa dan kapan struktur tersebut optimal untuk digunakan.
